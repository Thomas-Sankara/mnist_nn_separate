# mnist_nn_separate

此外，除了原教程和源代码里的注释，我个人添加的注释量其实占了绝大多数，包括关于pytorch的一些说明和python的某些语言特性的说明。<br>
主要参考了pytorch官方教程和《pytorch进阶之路（四）》这篇译文。链接如下：<br>
